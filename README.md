# rakuishi.nuxtjs

![](.github/hero.png)

> Nuxt.js project

## Build Setup

```bash
$ git clone https://github.com/rakuishi/rakuishi.com.git
$ git clone https://github.com/rakuishi/rakuishi.nuxtjs.git
$ cd rakuishi.nuxtjs
$ npm install
$ ln -s ../rakuishi.com/content content
$ npm run processmd
$ npm run dev
```
